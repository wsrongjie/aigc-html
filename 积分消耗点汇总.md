## 计费规则速查

| 操作类型 | 计费方式 |
|----------|----------|
| 首帧合成 / 重新合成 | 固定 **10** 积分 / 次 |
| 首帧重新生成（简单模式） | 固定 **10** 积分 / 次 |
| 视频生成 | **20** 积分 / 秒（按分镜时长计费） |
| AI 生图（场景） | 固定 **40** 积分 / 次 |
| 一键超分 | 固定 **5** 积分 / 次 |
| 语音试听 | **0**（浏览器 TTS，不走 API） |

---

## 弹窗样式说明

所有涉及积分的操作均通过二次确认弹窗拦截，复用 `.credits-confirm-overlay` + `.credits-confirm-modal` 组件。
文案统一格式：

> **此操作会消耗 🪙{N} 积分，请确定执行操作**

弹窗包含「取消」和「确定执行」两个按钮。
